# AfriStay - Hotel Locator for Africa

A beautiful, responsive hotel locator web application designed specifically for travelers exploring Africa.

## Framework

Vanilla JavaScript with HTML5 and Tailwind CSS (CDN)

## Features

- **Hotel Search**: Search hotels by location (city or country)
- **Advanced Filters**: Filter by country, price range, rating, and amenities
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop
- **Hotel Cards**: Beautiful hotel cards with images, ratings, prices, and amenities
- **Clean UI**: Modern, premium design with Clean Light theme
- **Real-time Filtering**: Instant results as you search and filter

## Preview

Open `index.html` in your browser to view the application.

## Theme

Clean Light theme with:
- Primary: Blue (#2563eb)
- Secondary: Green (#059669)
- Accent: Red (#dc2626)
- Background: White (#ffffff)
- Soft shadows and subtle borders for a premium feel

## Hotel Data

The application includes 18 sample hotels across 10 African countries:
- Kenya
- South Africa
- Egypt
- Morocco
- Tanzania
- Nigeria
- Ghana
- Rwanda
- Ethiopia
- Senegal

## Technologies

- HTML5
- Vanilla JavaScript (ES6+)
- Tailwind CSS (CDN)
- Font Awesome Icons (CDN)

## Usage

Simply open the `index.html` file in a modern web browser. No build process or installation required.
